Screen Reader Guide

Purpose of this Guide

This guide helps visually impaired users to navigate Archivehubs.com using screen readers such as NVDA, JAWS, VoiceOver(macOS/iOS), TalkBack(Android).

General Navigation Tips keyboard_arrow_right

  • keyboard_tab

    Use Tab to move between interactive elements (links, buttons, forms).

  • shift

    Use Shift + Tab to move backwards.

  • arrow_right_alt

    Use Arrow keys to navigate through text or menu lists.

  • space_bar

    Use Enter or Spacebar to activate buttons or open links.

  • h_mobiledata

    Use Heading Navigation (H key) to jump between sections or page headings.

Main Section & Landmark Regions keyboard_arrow_right

    Each major part of the site is labeled with ARIA landmarks to improve navigation:

  • panorama

    Banner

    Contains the logo, search bar, and navigation toggle.

  • navigation

    Navigation

    Main menu including Home, Drive, MyConnections, Monetization Hub, and Settings

  • content_paste

    Main Content

    Displays page-specific content (feeds, files, messages etc.).

  • view_sidebar

    Complementary (Sidebar)

    Suggestions, notifications, or additional features.

  • page_footer

    Footer

    Includes help, terms, privacy links, and social media.

Homepage keyboard_arrow_right

  • Use Heading (H) to skip through recent uploads, recommendations, and news.

  • Use Link (K or L) to explore file previews, user profiles, or CTA buttons like "Upload" or "Share".

Drive & File Management keyboard_arrow_right

  • Use Form (F) to access search input and filter tools.

  • Use Button (B) to locate actions like "Upload", "New folder", "Share", or "Download".

  • File items are marked with ARIA labels indicating type, size and last modified info.

MyConnections keyboard_arrow_right

  • Navigate to team activity feeds, messages, and shared files.

  • Voice prompts announce new messages, mentions, or collaboration updates.

Monetization Hub keyboard_arrow_right

  • Listen fro summaries of recent sales downloads, or earnings.

  • Tables with sortable headers (e.g. "File", "Earnings", "Date") are screen reader-accessible.

Interactive Features keyboard_arrow_right

  • Interctive charts, progress bars, and notifications are labeled with ARIA roles & descriptive text.

  • AI tagging and file categorization feedback are announced during uploads.

Settings & Profile keyboard_arrow_right

  • Navigate forms using F and fill out personal, security, and affiliate settings.

  • Dropdowns are accessible with keyboard arrows and Enter key.

Notifications & Activity keyboard_arrow_right

  • Notifications are grouped with list roles and described by type, sender, and timestamp.

  • Real-time updates are announced with live region ARIA tags.

Help & Support keyboard_arrow_right

  • Navigate to the Help Center via the main footer or Settings.

  • Use Heading (H) to move through FAQ topics and Link (L) to open relevant answers.

Accessibility Support keyboard_arrow_right

  • Report issues or request assistance at: archivehubs.com/accessibility.

  • Toggle Hight Contrast Mode and Font Adjustments via Settings > Accessibility.
